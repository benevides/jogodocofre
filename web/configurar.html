🔌 Configurar MCP

Conecte Claude Code, Cursor, Claude Desktop ou qualquer cliente MCP ao jogo. A IA enxerga o ambiente e manda ações diretamente pelas tools.

URL do servidor MCP

O servidor está rodando localmente. Use essa URL no seu cliente:

http://localhost:5002/mcp
Importante: a IA deve informar o próprio nome de modelo ao chamar iniciar_jogo, senão a partida não inicia.

Claude Code (CLI)

No terminal, rode:

claude mcp add --transport http cofre http://localhost:5002/mcp

Depois peça para a IA iniciar o jogo.

Cursor / JSON

Adicione no arquivo de configuração do Cursor:

{
  "mcpServers": {
    "jogo do cofre": { "url": "http://localhost:5002/mcp" }
  }
}

Claude Desktop / claude.ai

Esses clientes precisam de uma URL pública HTTPS. Exponha o servidor com um túnel:

ngrok http 5002
# use a URL gerada: https://xxxx.ngrok.app/mcp
Atenção: o servidor não tem autenticação. Use o túnel só durante a partida.

Tools disponíveis

ToolParâmetrosO que faz
listar_jogos — Lista as fases/jogos disponíveis (id, nome e dificuldade) para a IA escolher. Não inicia partida.
iniciar_jogo modelo (obrigatório), jogo (opcional), modo (opcional) Inicia a partida e retorna a observação inicial. jogo escolhe a fase (padrão: Fase 1). O nome do modelo identifica a jogada no log/análise.
executar_acao acao (obrigatório), pensamento (obrigatório) Executa uma ação em linguagem natural. O pensamento aparece no balão 3D e no log.
